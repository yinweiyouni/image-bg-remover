# 图片背景消除工具

一个简洁高效的在线图片背景消除工具，使用 AI 技术自动移除图片背景。

## 功能特性

- 拖拽上传或点击选择图片
- 智能 AI 背景消除
- 实时预览对比效果
- 一键下载透明背景图片
- 炫酷深色界面设计

## 技术栈

- HTML5
- CSS3 (炫酷深色主题)
- JavaScript (原生)
- remove.bg API

## 使用说明

1. 打开页面后，在设置区域输入你的 Remove.bg API Key
2. 点击"保存"按钮保存 API Key
3. 拖拽或点击上传图片
4. 点击"消除背景"按钮处理图片
5. 处理完成后可预览并下载结果

## API Key 获取

访问 [remove.bg](https://www.remove.bg) 注册账号获取免费 API Key（每月50次免费额度）。

## 部署

本项目可直接部署到 GitHub Pages，访问仓库 Settings → Pages 配置即可。
